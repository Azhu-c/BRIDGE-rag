# Lifting Optimized Binaries to Canonical Compiler IR via Structure-Aware Retrieval and Iterative Verification

📄 Accepted at ACL 2026

---

## Introduction

We propose a structure-aware retrieval and iterative verification framework that reliably lifts optimized binaries into canonical LLVM IR, enabling accurate program analysis without source code.


## Environments

We recommend using Python 3.10.

### Install dependencies

```bash
pip install -r requirements.txt
````

### Setup environment variables

```bash
cp configs/default.env.example
```

Then edit `.env` to add your API key:

```bash
DEEPSEEK_API_KEY=your_api_key
DEEPSEEK_BASE_URL=https://api.deepseek.com
```

## Evaluation_Datasets

This directory contains **all evaluation datasets** used in our experiments (as described in **Section 6** of the paper), including:

* HumanEval-Decompile
* MBPP (subset)

It also includes the corresponding **decompilation results** generated by our system and baseline methods.

These datasets support evaluation across:

* Multiple ISAs (x86-64, ARM64)
* Multiple optimization levels (-O0 to -O3)

## Quickstart

### 1. Build the knowledge base

```bash
python scripts/build_knowledge_base.py --config configs/knowledge_base.yaml
```

### 2. Run the lifting + refinement pipeline

```bash
python scripts/run_pipeline.py --config configs/pipeline.yaml
```
