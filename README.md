# BRIDGE-rag

**Appendix** includes supplementary experiments, showing the performance under **different optimization levels across various input lengths**, and a **decompilation demo** illustrating the effects of RAG enhancement and iterative refinement.

**Evaluation Datasets** contains all evaluation datasets used in the experiments (as described in Section VI.A of the paper).

**Prompts** contains all prompts used in the BRIDGE paper:
 - **LLVM_IR_Generation_Prompt.md** is used for the initial generation of LLVM IR (as described in Section V.A of the paper).
 - **Refinement_Prompt_static.md** is used for refining static errors(as described in Section V.B of the paper).
 - **Refinement_Prompt_dynamic.md** is used for refining dynamic errors(as described in Section V.B of the paper).
