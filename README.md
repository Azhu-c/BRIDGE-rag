# BRIDGE-rag
The prompts used in the BRIDGE paper are as follows:

 - **LLVM_IR_Generation_Prompt.md** is used for the initial generation of LLVM IR (as described in Section V.A of the paper).
 - **Refinement_Prompt_static.md** is used for refining static errors(as described in Section V.B of the paper).
 - **Refinement_Prompt_dynamic.md** is used for refining dynamic errors(as described in Section V.B of the paper).
