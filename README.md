# BRIDGE-rag

This repository contains the supplementary materials for the **BRIDGE** paper, including evaluation datasets, prompts, and additional experimental results.

### **Appendix/**

This directory provides **supplementary experiments and demonstrations**, including:

* The **performance comparison** under different optimization levels across varying input lengths.
* A **decompilation demo** that showcases the effects of **RAG-based enhancement** and **iterative refinement**.

### **Evaluation_Datasets/**

This directory contains **all validation datasets** used in the experiments (as described in **Section VI.A** of the paper). These datasets are used for evaluating the model’s effectiveness under different scenarios and configurations.

### **Prompts/**

 - **LLVM_IR_Generation_Prompt.md** is used for the initial generation of LLVM IR (as described in Section V.A of the paper).
 - **Refinement_Prompt_static.md** is used for refining static errors(as described in Section V.B of the paper).
 - **Refinement_Prompt_dynamic.md** is used for refining dynamic errors(as described in Section V.B of the paper).
